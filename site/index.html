Self-Hosted Homelab & Observability Platform
● Live

A production-grade homelab running on a Raspberry Pi 4 (Ubuntu Server) — fully managed as code, monitored, and recoverable from a bare OS in one command. This website is served from it.

  • 8 containerised services behind a Caddy reverse proxy, accessed privately over a WireGuard VPN (Tailscale) — no ports open on the router
  • Public portfolio site exposed via Cloudflare Tunnel — no home IP exposure, works behind CGNAT
  • Full observability: Prometheus + Grafana (metrics), Loki + Promtail (logs), Alertmanager (alerts to phone), Uptime Kuma (uptime)
  • Entire stack managed as code with Ansible — reproducible from bare OS; GitHub Actions CI validates every change
  • Automated backups with restic and a tested disaster-recovery runbook
  • Self-hosted Git (Gitea) mirrors configs to GitHub; self-hosted password manager (Vaultwarden)
